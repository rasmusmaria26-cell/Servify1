# Servify - Verified Local Service Providers

Servify connects you with verified local service providers for electronics repair, mechanical services, and home maintenance. AI-powered diagnosis, real-time tracking, and transparent pricing.

## Project info

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting Started

### Prerequisites
- Node.js & npm installed

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Deployment

You can deploy this project to Vercel, Netlify, or any other static hosting provider.

### Vercel
1. Push to GitHub.
2. Import project in Vercel.
3. Add Environment Variables (`VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`).
4. Deploy.
